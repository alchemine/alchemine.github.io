# CLI
## Create Bucket
```bash
aws s3api create-bucket \
--bucket alchemine-image \
--region ap-northeast-2 \
--create-bucket-configuration LocationConstraint=ap-northeast-2
```

## Put Object
```bash
aws s3api put-object --bucket alchemine-image --key 1.jpg
```

## Code
### 1. Set
{% highlight python linenos %}
def solution(phone_book):
    nums = set(phone_book)
    for num in phone_book:
        tmp = ""
        for n in num[:-1]:
            tmp += n
            if tmp in nums:
                return False
    return True
{% endhighlight %}

## Complexity
$O(n)$

- $n$ = len(phone_book)

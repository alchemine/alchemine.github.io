## Code
{% highlight python linenos %}
def solution(prices):
    answer = []

    n = len(prices)
    for i in range(n):
        pivot = prices[i]
        for j in range(i, n):
            if prices[j] < pivot:
                answer.append(j-i)
                break
        else:
            answer.append((n-1)-i)
    return answer
{% endhighlight %}


## Complexity
$O(n^2)$

- $n$ = len(prices)

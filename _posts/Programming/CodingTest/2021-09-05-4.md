## Code
{% highlight python linenos %}
def solution(citations):
  sorted_citations = sorted(citations, reverse=True)
  for i in range(len(sorted_citations)):
    if sorted_citations[i] <= i:
      return i
  return len(sorted_citations)
{% endhighlight %}


## Complexity
$O(nlogn)$

- $n$ = len(citations)

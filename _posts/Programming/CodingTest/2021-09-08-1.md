## Code
{% highlight python linenos %}
def solution(brown, yellow):
    answer = []

    for c in range(1, yellow + 1):
        d = yellow / c
        if d != int(d) or c > d:
            continue
        d = int(d)
        if (c+2)*(d+2)-yellow == brown:
            return [d+2, c+2]

    return answer
{% endhighlight %}


## Complexity
$O(n)$
- $n$ = len(yellow)

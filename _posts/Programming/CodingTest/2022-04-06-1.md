## Code
{% highlight python linenos %}
def solution(s):
    for num, name in enumerate(['zero', 'one', 'two', 'three', 'four', 'five', 'six', 'seven', 'eight', 'nine']):
        s = s.replace(name, str(num))
    return int(s)
{% endhighlight %}


## Complexity
$O(|s|)$

# 1. FastAPI의 철학
1. High performance
2. Easy to learn
3. Fast to code
4. Ready for production

# 2. FastAPI의 특징
1. Starlette framework 기반으로 인한 비동기 API 서버 지원
2. Pydantic을 이용한 data validation 지원
3. **OpenAPI를 사용하여 API Swagger를 자동으로 생성**
4. JS, GO와 비교하여 성능, 부하 테스트에서 크게 밀리지 않음

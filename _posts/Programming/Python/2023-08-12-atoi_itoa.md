<!--more-->

- `ord(c)`: 유니코드 문자를 숫자로 바꾸는 함수 (`atoi`)
- `chr(i)`: 숫자를 유니코드 문자로 바꾸는 함수 (`itoa`)

```python
assert ord('1') == 49 and chr(49) == '1'
assert ord('A') == 65 and chr(65) == 'A'
assert ord('a') == 97 and chr(97) == 'a'
assert ord('€') == 8364 and chr(8364) == '€'
```
